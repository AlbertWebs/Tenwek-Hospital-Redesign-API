## 5. Rules applied

- **External links:** Open in new tab (`target="_blank" rel="noopener noreferrer"`), with “(External)” or “↗” where appropriate.  
- **No duplicates:** One canonical link per destination; cross-links only where hierarchy or UX requires.  
- **Hierarchy:** Featured centres and main service groups stay grouped; Quick Links for high-traffic actions (Book, Donate, Contact, etc.).  
- **Scalable:** Footer groups are consistent with IA; new services or centres can be added under the same groups.
